# Build-it-Bigger

## Features available
- Paid Version (No Ads)
- Free Version
- GCE
- Java Library
- Android Library
- Has Spinner Loading
- Interstitial Ad
- Instrumental Test to test Async Task
- Gradle task for full testing

## Instructions to run Gradle task
From root of the project, run (Windows)

    gradlew rFT

The key inside `strings.xml` must be replaced for displaying Ads
